# GitHub & Git Workflow

This workflow provides common Git and GitHub commands. By using this workflow, you agree to auto-accept these commands.

// turbo-all

## Status and Information
- `git status` - Check the current state of the repository
- `git log -n 5` - View the last 5 commits
- `git diff` - View unstaged changes

## Committing Changes
- `git add .` - Stage all changes
- `git commit -m "Your message"` - Commit staged changes

## Remote Operations
- `git pull` - Fetch and merge changes from the remote
- `git push` - Push local commits to the remote
- `git fetch` - Download objects and refs from another repository

## Branch Management
- `git branch` - List local branches
- `git checkout -b branch-name` - Create and switch to a new branch
- `git merge branch-name` - Merge a branch into the current one
